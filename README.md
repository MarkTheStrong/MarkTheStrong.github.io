# MY WEBSITE

Its my take on a parallax scroll where you go into the tent. IT has a home page with 3 different services

## FEATURES

-> Designs for hopefully all devices
-> Clean UI
-> Mobile friendly ult with a hamburger menu
-> Contact form (for emails from interested ppl)
-> Smooth transitions and animations
-> Avail on multiple different browsers WOOO

## PAGES

1. **Homepage (index.html)**
    + Parallax scroll
    + Features all the pages as options 
2. **About page (aboutMe.html)**
    + My resume information and such
    + clickable expanding interface
3. **Repository Page(Repository.html)**
    + Contains my files from previously done work
    + Downloadable clickable interface
4. **Contact (contactMe.html)**
    + Contains a contact form for ppl to message me
    + Linked to my personal form account
    + Pressable linkedin image


## JAVASCRIPTS

1. **script.js**
    + Contains the movement and animation of pages
    + Applies content to each page
    + Parallax scroll is done here
2. **content.js**
    + Content for the pages named and typed there
    + Future multiple language section in this file

## STYLES

1. **styles.css**
    + Contains the styles and animations of the pages
2. **colors.css**
    + Contains all of the colors to choose from on the pages
    + Makes it easier to do big changes color wise

## Technology Used

- HTML5
- CSS3
- JavaScript (ES6+)
- CSS Grid and Flexbox
- Responsive Design
